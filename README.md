# Regrese
* Smyslem je predikce nových hodnot (neznámých) využívající ono proložení známých hodnot přímkou.
* Spadá to kategorie učení s učitelem.
* Rozlišovány jsou 2 základní typy regrese
  - Lineální regrese
  - Logická regrese
